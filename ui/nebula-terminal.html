The emulator engine, BIOS, and a real Linux kernel are downloaded from a CDN. Browsers block these cross-origin downloads when the page is opened from a file:// path — which is the usual cause. Run it from a local web server instead:

cd your-repo-folder
python3 -m http.server 8000
# then open  http://localhost:8000/ui/nebula-terminal.html

It also runs as-is on GitHub Pages, since that's served over http.
